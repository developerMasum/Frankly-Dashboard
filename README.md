## Run Locally

Clone the project

```bash
  git clone https://github.com/developerMasum/Frankly-Dashboard
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm  start
```
